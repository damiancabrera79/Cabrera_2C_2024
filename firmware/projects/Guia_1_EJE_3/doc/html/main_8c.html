void app_main ( void  )

Llama a la función que controla el comportamiento del LED según la configuración.

Esta función toma la configuración especificada en la estructura 'tipo_led' y realiza la operación correspondiente sobre el LED. El LED será encendido, apagado,
o alternará su estado basado en el modo de operación y los ciclos especificados.

Parámetros
&ledsPuntero a la estructura 'tipo_led' que contiene la configuración del LED.
Nota
Asegúrate de que 'LedsInit()' haya sido llamada antes de usar 'funcion_leds()'.

◆ funcion_leds()

void funcion_leds ( tipo_led *  est_led)

Función para controlar el estado de los LEDs.

Esta función controla el estado de un LED según el modo especificado (ON, OFF, TOGGLE) y realiza la operación durante el número de ciclos indicado con el periodo especificado.

Parámetros
est_ledPuntero a una estructura tipo_led que contiene la configuración del LED.
Nota
Dependiendo del modo, el LED puede permanecer encendido, apagado, o alternar entre encendido y apagado.
