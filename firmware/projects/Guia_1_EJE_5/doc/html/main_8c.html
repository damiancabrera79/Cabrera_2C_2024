void configurarGpios ( uint8_t  bcd_digito,
gpioConf_t  gpio_conf[4] 
)

Configura y cambia el estado de los pines GPIO según el dígito BCD.

Esta función toma un dígito BCD y un vector de configuraciones de GPIOs. Cada bit del dígito BCD se usa para definir el estado del correspondiente pin GPIO: si el bit es 1, el GPIO se configura en el estado alto; si el bit es 0, el GPIO se configura en el estado bajo.

Parámetros
bcd_digitoDígito BCD que define el estado de los pines GPIO
gpio_confVector de configuraciones de GPIOs donde se almacenan los pines y sus direcciones.
Nota
El tamaño del vector gpio_conf debe ser al menos de 4 elementos, uno para cada bit del dígito BCD (b0 a b3).
