Guia 2 - Ejercicio 2 1.0
Este programa controla el encendido y apagado de LEDs, la lectura mediante un sensor de distancia HC-SR04 y la visualización de la distancia en un display LCDITSE0803 utilizando un ESP32. El sistema utiliza Las interrupciones para el control de las teclas y para el control de tiempos Timers.
